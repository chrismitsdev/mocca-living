# mocca-living

Repo of the Mocca Living website.

You can take a look at the website [here](https://moccaliving.com).

**Core project tools:**

- [Next.js](https://nextjs.org/) as the meta-framework of choice
- [Tailwind CSS](https://tailwindcss.com/) for all styling purposes
- [next-intl](https://next-intl-docs.vercel.app/) as the i18n solution
- [Radix UI](https://www.radix-ui.com/primitives) accessible primitives
- [Motion](https://motion.dev/) a popular animation library for the web
- [Leaflet](https://leafletjs.com/) JS library for mobile-friendly maps
- [React Leaflet](https://react-leaflet.js.org/) components for Leaflet maps
- [Tabler Icons](https://tabler.io/icons) a rich React icons library
- [Valibot](https://valibot.dev/) schema & data validation library
- [Resend](https://resend.com/home) API for sending transactional emails
